# NCM LMS V8 Server Edition

هذه نسخة خادم تجريبية تعمل بقاعدة بيانات SQLite، وتدعم الدخول من أكثر من جهاز إذا تم تشغيلها على سيرفر واحد.

## كلمات المرور الافتراضية
- الإدارة: `NCM@2026`
- HR: `HR@2026`
- المدرب: `Trainer@2026`

## التشغيل المحلي
1. ثبّت Node.js.
2. افتح المجلد في Terminal.
3. نفّذ:
```bash
npm install
npm start
```
4. افتح:
```text
http://localhost:3000
```

## البريد الإلكتروني
المنصة تعمل بدون بريد حقيقي وتطبع الرسائل في Console.
لتفعيل البريد الحقيقي انسخ `.env.example` إلى `.env` وأدخل إعدادات SMTP.

## الرفع على سيرفر
هذه النسخة ليست Netlify Static فقط. تحتاج بيئة Node.js مثل:
- Render
- Railway
- VPS
- Azure App Service
- خادم داخلي بالمركز
